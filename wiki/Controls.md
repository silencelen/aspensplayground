# Controls

Complete control reference for all platforms.

---

## Desktop Controls

### Movement

| Action | Primary | Alternative |
|--------|---------|-------------|
| Move Forward | W | ↑ Arrow |
| Move Backward | S | ↓ Arrow |
| Strafe Left | A | ← Arrow |
| Strafe Right | D | → Arrow |
| Jump | Space | - |
| Sprint | Left Shift (hold) | - |

### Combat

| Action | Control |
|--------|---------|
| Aim | Mouse Movement |
| Shoot | Left Mouse Button |
| Reload | R |
| Throw Grenade | G |
| Melee (if available) | V |

### Weapons

| Action | Control |
|--------|---------|
| Pistol | 1 |
| Shotgun | 2 |
| SMG | 3 |
| Assault Rifle | 4 |
| Sniper Rifle | 5 |
| Laser Gun | 6 |
| Grenade Launcher | 7 |
| Next Weapon | Scroll Up |
| Previous Weapon | Scroll Down |

### Interface

| Action | Control |
|--------|---------|
| Pause Game | Escape |
| Toggle Fullscreen | F11 |
| Open Settings | Pause → Settings |

### Developer/Debug (Single Player Only)

| Action | Control |
|--------|---------|
| Infinite Ammo | F6 |

## Mobile Controls

### Touch Zones

```
┌─────────────────────────────────────────┐
│                                         │
│   ┌─────────┐           ┌─────────┐     │
│   │ Joystick│           │  Look   │     │
│   │  (Move) │           │  Area   │     │
│   └─────────┘           └─────────┘     │
│                                         │
│   [Sprint]              [Jump] [Reload] │
│                              [Grenade]  │
│   ┌─────────────────────────────────┐   │
│   │  1  │  2  │  3  │  4  │  5  │ 6 │   │
│   │ Weapon Selection Bar            │   │
│   └─────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

### Touch Actions

| Action | How To |
|--------|--------|
| Move | Drag left joystick |
| Look Around | Swipe on right side of screen |
| Shoot | Tap right side OR dedicated fire button |
| Reload | Tap reload button |
| Jump | Tap jump button |
| Sprint | Hold sprint button |
| Switch Weapon | Tap weapon in bottom bar |
| Throw Grenade | Tap grenade button |

### Mobile Tips

- The game auto-fires when aiming at enemies (optional in settings)
- Use two thumbs: left for movement, right for aiming
- Weapon bar can be hidden in settings for more screen space
- Haptic feedback provides vibration on actions (if enabled)

## Accessibility Options

Available in Settings:

| Option | Description |
|--------|-------------|
| Sensitivity | Adjust mouse/touch look sensitivity |
| Invert Y-Axis | Invert vertical look |
| Auto-Fire (Mobile) | Automatically shoot when aiming at enemies |
| Reduce Motion | Disable screen shake and reduce animations |
| Haptic Feedback | Toggle vibration on mobile |

## Tips for Better Control

### Mouse Sensitivity
- Start at medium sensitivity
- Increase if you're under-aiming (always playing catch-up)
- Decrease if you're over-aiming (constantly overshooting targets)

### Keybinds
- Weapon keys (1-7) are faster than scrolling
- Learn to reload with R while moving
- Sprint-jump to cover distance quickly

### Mobile Optimization
- Play in landscape mode
- Enable "Do Not Disturb" to prevent notifications
- Disable auto-brightness for consistent visibility
